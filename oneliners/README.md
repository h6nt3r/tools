## Blind XSS install
```
wget "https://raw.githubusercontent.com/h6nt3r/tools/refs/heads/main/oneliners/bxsser.sh"
sudo chmod +x ./bxsser.sh
./bxsser.sh -h
```
## usage
```
./bxsser.sh -d "http://testphp.vulnweb.com"
```
```
./bxsser.sh -l "http://vulnweb.com"
```
